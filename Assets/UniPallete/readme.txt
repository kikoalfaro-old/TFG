1. Import package
2. Goto Window/UniPallete
3. Configure your pallete colors
4. Select objects to colorize
5. Click "Colorize" of any color
6. Repeat the same with other objects



Supported Objects for Colorization

Camera (backgroundColor)
Renderer (color) any renderer, ex: MeshRenderer
Light (color) any lighting, Point, Spot or Directional Light
Text (color) UI text
Image (color) UI images, any of them, in buttons or toggles etc.
RawImage (color)


Features:
Awesome UI
Color automatic save & load
Multiple themes
Multiple type of object colorization
